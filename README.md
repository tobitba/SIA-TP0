
# TP0 SIA - Análisis de Datos

## Introducción

Trabajo práctico orientativo para la materia Sistemas de Inteligencia Artificial con el
objetivo de evaluar la función de captura de un Pokemon.

[Enunciado](docs/SIA_TP0.pdf)

### Requisitos

- Python3
- pip3
- [pipenv](https://pypi.org/project/pipenv/)

### Instalación

Parado en la carpeta del tp0 ejecutar

```sh
pipenv install
```

para instalar las dependencias necesarias en el ambiente virtual

## Ejecución
Para ejecutar cada uno de los ejercicios del tp ejecutar
```
pipenv run python ejercicio_xx.py
```
Por ejemplo para correr el ejercicio 1a

```
pipenv run python ejercicio_1a.py
```
